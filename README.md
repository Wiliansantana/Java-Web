# Java-Web
Nesse projeto o objetivo foi montar um sistema de gestão para uma clínica, onde poderá ter um melhor controle de agendamento de horários para consultas, cadastrar novos médicos e pacientes etc. Para a criação desse projeto usamos a linguagem JAVA para Back-end, para Front end usamos um framework chamado PrimeFaces JSF, e para salvar nossos dados usamos o MySQL Workbench.
